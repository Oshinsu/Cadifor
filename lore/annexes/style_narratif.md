# STYLE D'ECRITURE ET NARRATION

---

## VOIX NARRATIVE

| Attribut | Description |
|----------|-------------|
| **Registre** | Historien-chroniqueur passionne, entre analyse academique et admiration contenue |
| **Perspective** | D'un "historien" qui commente la dynastie comme un evenement civilisationnel |
| **Ton** | Solennel mais pas precieux, capable de basculer dans l'emotion sans sentimentalisme |
| **Distance** | Assez proche pour etre touche, assez loin pour analyser |

### Tics stylistiques
- **Phrases courtes percutantes** alternant avec des **periodes longues analytiques**
- **Enumeration a tirets** pour les listes de qualites
- **Questions rhetoriques** : "Voila pourquoi..." / "C'est cela, au fond, qui..."
- **Formules definitives** : "Ce n'est pas un detail ; c'est une decision civilisationnelle"
- **Points-virgules** comme ponctuation de nuance
- **Italiques** pour les concepts-cles et les citations

## QUATRE REGISTRES ALTERNES

### 1. Chronique Imperiale
- Style "Gazette de Lordaeron"
- Voix medievale, solennelle
- Dates precises, titres complets
- *"En l'an 767 de l'ere commune, Andrea, dite la Victorieuse, monta sur le trone..."*

### 2. Roman / Scene
- Dialogues immersifs entre personnages
- Descriptions physiques, atmospheriques
- Exemple : Rose et Arwyn au bar, le duel amical
- *"On ne la voit jamais." La taverne sentait le bois mouille et la biere tiede.*

### 3. Analyse Strategique
- Decomposition geopolitique
- Lecture de screenshots CK2
- Chiffres, pourcentages, comparaisons
- *"27 199 vs 15 067 : la bataille d'Allerton est un ecrasement numerique."*

### 4. Game Design
- Traduction en termes WoW Classic
- Tiers, sorts, equipement, builds
- *"Frostfire Regalia T3 + Atiesh = le build le plus absurde du Vanilla."*

## LA FORMULE DYNASTIQUE

> *"La grandeur des Cadifor ne vient pas d'un seul monstre. Elle vient du fait que plusieurs monstres differents regnent assez longtemps pour transformer leur nature en institutions."*

Cette phrase est la **cle de voute** de tout le lore. Elle contient :
1. **"Plusieurs monstres differents"** : chaque souverain est exceptionnel, mais differemment
2. **"Regnent assez longtemps"** : la duree est une arme (51 ans pour Viki, 57 pour la Juste, 50 pour Benjamin)
3. **"Transformer leur nature en institutions"** : la grandeur individuelle devient systeme
