# Cercle Cenarien

## Principe / lecture

Le `Cercle Cenarien` est une autorite druidique transnationale plus qu'un royaume classique. Les sources l'installent comme composante majeure de l'ordre kalimdorien, au contact des `Kaldorei`.

## Identite

| Champ | Valeur |
|-------|--------|
| Statut | Contact limite avec le Haut Royaume |
| Nature | Autorite druidique transnationale |
| Region principale | `Kalimdor` |
| Place geopolitique | Co-domine la majeure partie de `Kalimdor` avec les `Kaldorei` |

## Points structurants

- Les correctifs tardifs parlent explicitement d'un ensemble `Empire kaldorei / cercle cenarien`.
- Les syntheses le decrivent comme une puissance de cycles, de regeneration et de terrain plutot que de frontiere stricte.
- Les relations avec les `Cadifor` sont possibles localement, mais restent peu solubles dans les categories imperiales.
- La documentation disponible reste surtout de lecture geopolitique generale, sans dossier institutionnel fin.
