# ECONOMIE DE L'EMPIRE CADIFOR

---

## LE "DEAL D'ABERTHOL" — Architecture economique fondatrice

### Principe
Aberthol a transforme les mariages de ses filles en **infrastructure commerciale** :

| Fille | Position | Fonction economique |
|-------|----------|-------------------|
| **Gwen** | Reine de Kul Tiras | Routes maritimes + tarifs favorables |
| **June** | Reine de Stormgarde | Routes terrestres + tarifs favorables |
| **Andrea** | Darrow (reseau) | Maestro operationnel : espionnage, tresorerie secrete |

### Mecanismes

| Outil | Description | Operateur |
|-------|-------------|-----------|
| **Routes favorables** | Corridors commerciaux preferentiels Kul Tiras <> Cadifor | Gwen |
| **Tarifs avantageux** | Droits de douane reduits pour les marchandises Cadifor | Gwen + June |
| **Guildes d'information** | Espionnage commercial en Sylve de l'Est et au-dela | Andrea |
| **Forestiers renegats** | Elfes de sang hors Quel'Thalas, agents de terrain | Andrea |
| **Espions en Lordaeron** | Reseau de renseignement dans le royaume voisin | Andrea |
| **Comptoirs sous faux pavillon** | Societes ecrans pour le commerce indirect | Andrea |
| **Tresorerie secrete** | Fonds hors comptes officiels du duche | Andrea |
| **Raids aux Hinterlands** | Reliques troll, or, metaux precieux | Andrea |

### Impact a long terme
- La richesse accumulee finance l'expansion de l'Erudite (676-710)
- Les alliances commerciales survivent 200+ ans apres leur creation
- Le sang Trollbane (via Stormgarde/June) produit la Victorieuse
- Le modele est replique : chaque generation ajoute des routes et des comptoirs
