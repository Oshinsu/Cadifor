## LA DOCTRINE MARJORY

> *"Un palais marjoryen est un lieu ou chaque matiere a ete forcee a devenir plus intelligente qu'elle-meme."*

La doctrine Marjory est la philosophie architecturale creee par l'Imperatrice Marjory (909-943). Elle transforme chaque element du batiment en un **artefact fonctionnel** : la matiere n'est pas decorative, elle est **active**.

---

## PALAIS IMPERIAL PRINCIPAL — Stormwind

### Fonction
Represente **l'Empire**. Le pouvoir a l'echelle continentale.

### Construction en 7 strates de souverainete

| Strate | Fonction | Acces |
|--------|----------|-------|
| 1. **Publique** | Cours, jardins, entrees monumentales | Tout le monde |
| 2. **Nobiliaire** | Salles de reception, salons d'audience | Aristocratie, ambassadeurs |
| 3. **Militaire** | Salle de guerre, casernes d'elite | Commandement militaire |
| 4. **Liturgique** | Chapelles, autels de la Lumiere | Clerge, paladins |
| 5. **Decision** | Salle du Conseil, bureaux imperiaux | Conseil restreint |
| 6. **Secret** | Services de renseignement, salles blindees | Maitres espions |
| 7. **Dynastique** | Appartements imperiaux, cryptes familiales | Famille Cadifor uniquement |

### Materiaux

| Materiau | Utilisation | Propriete |
|----------|-------------|-----------|
| **Marbre veine** | Murs, sols, colonnes | Pierre claire a inclusions de mica et poussiere stellaire |
| **Nervures de mithril** | Structure, decoration, runes | Conducteur magique, renforcement |
| **Joints runiques** | Jonctions entre blocs | Traces magiques de consolidation |
| **Cristal mineral** | Verrieres | Filtrage de lumiere, projection de motifs |

### Salle du Trone

| Element | Description |
|---------|-------------|
| **Volume** | Nef immense, colonnes de marbre pale |
| **Voute** | Ecrasante par justesse — pas par taille, mais par perfection geometrique |
| **Verriere** | Cristal mineral filtrant une lumiere architecturale |
| **Trone** | Pierre blanche veinee gris-or, accotoirs geometriques |
| | Insertions de mithril noirci |
| | Runes de garde actives |
| **Acoustique** | La voix du trone porte loin. Les chuchotements des suppliants se perdent. |

## PALAIS SECONDAIRE — New Avalon

### Fonction
Represente **la MAISON** qui possede l'Empire. L'intimite du pouvoir.

### Style
- Plus **sombre**, plus **soyeux**, plus **dense** que Stormwind
- Pas de grandeur ecrasante : une elegance feutree

### Usages
- Conversations vraies (pas de protocole complet)
- Receptions restreintes (famille, allies proches)
- Decisions molles (celles qui ne se prennent pas en salle du trone)
- Soirees confidentielles

### Materiaux

| Materiau | Utilisation |
|----------|-------------|
| **Velours de nuit** | Tentures, sieges, rideaux |
| **Fourrures blanches** | Tapis, accoudoirs |
| **Cuir souple** | Reliures, coffrets, fourreaux |
| **Poignees emaillees** | Portes, meubles |
| **Cristaux-lampes** | Lumiere d'accord — s'adapte a l'ambiance |

## INNOVATIONS TECHNOLOGIQUES

### Lampes Marjoryennes

| Composant | Description |
|-----------|-------------|
| **Base** | Pierre de lune noire |
| **Cage** | Cristal double — deux couches transparentes |
| **Runes** | Micro-runes de diffusion gravees dans le cristal |
| **Coeur** | Goutte d'huile minerale suspendue par tension magique |
| **Fonctionnement** | La lumiere **varie selon la densite de conversation** dans la piece |

> Une lampe marjoryenne ne s'allume pas : elle *ecoute* la piece et repond.

### Table de salle a manger

| Propriete | Description |
|-----------|-------------|
| **Materiau** | Pierre sombre incrustee de veines de mithril et silice lunaire |
| **Maintien thermique** | La table garde les plats a temperature ideale |
| **Anti-tache** | Surface repousse les liquides |
| **Memoire runique** | Se souvient du placement optimal des convives |

### Portes

| Element | Description |
|---------|-------------|
| **Materiau** | Bois dense |
| **Ferrures** | Argent noirci |
| **Charnieres** | Silencieuses — aucun grincement |
| **Joints** | Runiques — etancheite sonore et magique |
| **Inscriptions** | Memoire — la porte "sait" qui est autorise |

> *On ne passe pas une porte. On traverse une decision ancienne.*

### Ecuries imperiales

| Element | Description |
|---------|-------------|
| **Architecture** | Nefs claires en pierre blonde |
| **Sols** | Draines et chauffes — pas d'humidite |
| **Plaques emaillees** | Une par monture : nom, age, lignage, temperament, blessures |
| **Noeuds runiques** | De calme — les montures restent sereines |
| **Selleries** | Plus parfaites que des tresors de duches |
| **Lumiere** | Basse, modulee, pour ne jamais blesser l'oeil nocturne |
| **Crochets d'equipement** | Classes selon ordre d'intervention |

> Une ecurie imperiale marjoryenne doit sentir le foin noble, le cuir juste, la proprete animale, jamais l'abandon.

## GALERIE DYNASTIQUE

### Salle du trone des souvenirs

Lieu ou :

- armures des ancetres
- bannieres de campagne
- lames
- lettres
- sceaux
- portraits
- fragments de pierre de champs de bataille

sont mis en ordre pour former une **theologie de la lignee**.

Ce lieu doit etre plus impressionnant qu'un tresor.

Pourquoi :

- un tresor montre ce qu'on possede
- une galerie dynastique montre ce qui nous a rendus capables de posseder

## ARMURES

### Armure de Marjory
- **3 etats** : presence / campagne / ceremonie
- **Materiau** : Plaque ivoire metallique
- **Details** : Mithril brode dans les lisieres, micro-runes de tenue et de silence
- Chaque etat a son propre niveau de complexite et de decoration

### Armure de Rose
- Plus **mobile** que celle de Marjory
- Plaques fines articulees
- Couleurs : ivoire, argent pale, gris de perle
- Inscriptions **necro-arcanes cachees** — visibles seulement sous certains angles de lumiere
- Plus legere mais plus protegee (par la magie)

## ROBES

### Robes de Marjory
- Tissus **lourds**, couleurs profondes :
  - Bleu cendre
  - Noir vert
  - Gris imperial
- Broderies **presque invisibles** — il faut regarder de pres
- Fils de mithril dans les ourlets
- **Polyvalence** : permettent salon, bibliotheque, magie, et presque la guerre

### Robes de Rose
- Plus **fluides** que celles de Marjory
- Gris de perle, noirs translucides
- Peut integrer **talismans et artefacts** sans les rendre visibles
- Memes polyvalence : salon / bibliotheque / magie / presque la guerre
- La difference : les robes de Rose sont **plus legeres** mais plus chargees en enchantements invisibles
