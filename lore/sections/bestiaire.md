# BESTIAIRE ET MONTURES DE L'EMPIRE

---

## PRINCIPES

1. Les montures rares = **cadeaux/prets** aux vassaux fideles et eminences de cour
2. Monture volante ou dragon = **prestige**, PAS utilitarisme (un mage rang X les one-shot)
3. Rose et Marjory utilisent principalement la **teleportation** — pas besoin de monture
4. Le nombre de montures exotiques est **strictement limite** pour preserver le prestige
5. Le **cheval** reste la monture de verite de la maison Cadifor : le centre moral n'est pas l'exotisme, mais la lignee de marechaux

## NOTES SUR LES NIGHTSABERS

Le sujet des nightsabers est un **point de friction diplomatique** majeur :
- Les elfes de la nuit (Kaldorei) sont la source originelle
- Quel'Thalas (Anasterian, toujours vivant) est **tres jaloux** de voir des humains sur des nightsabers
- Chaque nightsaber dans les ecuries imperiales est un **message geopolitique**
- Andrea la Victorieuse a probablement recu les premiers comme cadeaux diplomatiques

## LES ECURIES IMPERIALES

Sous la doctrine Marjory, les ecuries sont traitees comme des **palais animaux** :

| Element | Description |
|---------|-------------|
| Architecture | Nefs claires en pierre blonde |
| Sols | Draines et chauffes — zero humidite |
| Identification | Plaque emaillee par monture : nom, age, lignage, temperament, blessures |
| Confort | Noeuds runiques de calme — les montures restent sereines |
| Personnel | Ecuyers specialises par espece |
| Sellerie | Crochets d'equipement classes selon ordre d'intervention |
| Doctrine | Une selle de haut rang est deja une tradition de guerre |
