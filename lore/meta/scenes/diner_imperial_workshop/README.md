# DINER IMPERIAL — Atelier des 4 versions

---

## Statut

Ce dossier n'est pas le canon.

Le canon continu du diner reste :

- `lore/meta/scenes/diner_imperial.md`

Ici se trouvent les materiaux d'orfevrerie necessaires pour produire une seule scene exceptionnelle a partir de plusieurs etats d'ecriture.

---

## Principe

Le diner imperial ne peut pas etre traite comme une simple scene a corriger.

Il est :

- une scene de civilisation
- une scene de transmission mere / fille
- une scene de doctrine imperiale
- une scene possible d'ouverture de toute l'oeuvre

Il faut donc accepter que plusieurs versions disent juste des choses differentes.

La methode retenue est la suivante :

- `V1` : seuil, magnetisme, rythme d'entree, phrases-bangers
- `V2` : scene continue la plus exploitable d'un trait
- `V3` : coupes, dialogues, copresence souveraine, vitesse
- `V4` : contexte, coherence, ancrage geopolitique, canon actuel

Le but n'est pas de choisir une version contre les autres.
Le but est de monter une seule scene dont aucune des versions existantes n'epuise seule la hauteur.

---

## Fichiers du dossier

- `source_pdf.md` : cartes des plages-source verifiees dans `Lore cadifor 3.pdf`
- `V1_seuil.md` : ce qui est irremplacable dans l'ouverture et le rythme
- `V2_scene_continue.md` : scene-source la plus complete avant canonisation
- `V3_dialogues.md` : gestes de coupe, lignes, repliques, logique de copresence
- `V4_canon.md` : rappel de l'etat canon actuel
- `phrases_fortes.md` : phrases classees par fonction dramatique
- `notes_auteur.md` : contraintes, interdits, et exigences de ton

---

## Regle de montage

Une phrase ne survit au montage que si elle remplit au moins trois conditions :

1. elle sonne comme Marjory ou Rose, pas comme une note explicative
2. elle garde sa force une fois replongee dans le flux de la scene
3. elle augmente la hauteur de la scene sans casser la coherence du monde

Si deux formulations se valent, on garde celle qui :

- ouvre le plus d'implications avec le moins de poids
- laisse le lecteur comprendre sans etre aide
- donne envie de relire la phrase plutot que de la survoler

---

## Usage

Le bon ordre de travail est :

1. lire `source_pdf.md`
2. relire `V1_seuil.md`
3. relire `V3_dialogues.md`
4. verifier `V4_canon.md`
5. puiser dans `phrases_fortes.md`
6. ne retoucher `diner_imperial.md` qu'apres

Ce dossier doit rester un atelier de precision.
Pas un cimetiere de variantes.
