# SCREENSHOTS CK2 — LOT FICHES PERSONNAGES

---

## But

Ce fichier recense toutes les **fiches personnages** (character sheets) visibles dans les screenshots CK2 fournis.

Format par personnage : identification, date, âge, stats principales, traits, ressources, certitude.

---

## Echelle de certitude

- **C3** : valeur lisible directement, sans ambiguïté
- **C2** : très probable, lecture en grande partie sûre
- **C1** : hypothèse prudente, valeur partiellement lisible
- **C0** : non affirmable
