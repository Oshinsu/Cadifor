## Source

- `lore cadifor original.pdf`
- Total de reference : `565 pages`

Ce fichier suit non pas un simple pourcentage de lecture,
mais la **cartographie des zones a haut rendement** du PDF original.

Pourquoi :

- le document est plus brut
- il contient des ecrans commentes, des corrections utilisateur et des proses IA melangees
- il est souvent plus exact dans la donnee brute que `Lore cadifor 3.pdf`
- il exige donc un tri interne plus fin

---

## Regle de lecture

Dans le `PDF original`, toutes les lignes ne se valent pas.

Ordre interne de priorite :

1. `corrections utilisateur` explicites
2. `donnees d'ecrans CK2` ou deductions immediates depuis ces ecrans
3. `resumes reformules` qui ne contredisent pas les deux premiers niveaux
4. `proses IA developpees` seulement si elles ne deformaient pas deja la charpente

Autrement dit :

> *Le PDF original est prioritaire comme gisement, mais il ne doit pas etre avale en bloc ; il doit etre trie.*

## Ce que ces sondages ont deja appris

### 1. Le PDF original est la vraie mine des corrections dures

Le tout debut contient deja :

- la correction `Ulf` -> faux
- la correction `Sylve de l'Est` hors `Azeroth` avant l'heritage d'`Andrea II`
- la clarification du mariage `Llane x Andrea Cadifor`
- la structure exacte `screenshots -> correction utilisateur -> reformulation`

### 2. Les blocs romanesques sont melanges

Exemple type :

- une premiere prose IA trop libre
- une correction utilisateur
- une seconde prose mieux recalee

Il faut donc toujours chercher si une correction explicite vient **apres** une belle page apparemment convaincante.

### 3. Le haut Empire y est tres riche

Les pages tardives du PDF original contiennent des materiaux precieux sur :

- fiscalite
- villes
- seigneuries
- credit
- cartographies administratives
- centres secondaires de souverainete

Ce n'est pas seulement un PDF des origines.

## Rendements par type

### Tres fiable

- user messages correctifs
- listes territoriales explicites
- dates et stats donnees par l'utilisateur
- lineages lus sur les screens

### Fiable mais a reexprimer

- analyses de seigneuries
- descriptions d'economie regionale
- schemas institutionnels de haut empire

### A manier avec prudence

- grandes proses biographiques
- justifications politiques improvisees pour des evenements CK2
- noms secondaires non verifies
- genealogies trop belles, trop lisses, ou trop dramatiques

### A ne pas reimporter sans controle

- `Bradney` ou `Ariana` comme participants a la guerre civile d'`Andrea II` (ils existent mais dans d'autres contextes)
- `Hrothar` (erreur de nom pour Johnson), `Felita` (erreur de nom/source)
- `Anastasia Wrynn` comme personnage majeur de cette periode
- toute lecture d'un `Azeroth` faible avant la centralisation cadiforienne
